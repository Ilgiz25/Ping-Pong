## Ping-pong
Сюжет основан на реальных событиях





## Что имеем:

 - 2 спрайта ракетки 
 - Спрайт мячика
 


## Цель игры
Развить реакцию игроков и сократить досуг.




## Детали
- две ракетки управляющиеся с клавиатуры
- мяч двигающийся по сцене автоматически
- проигрывет игрок пропустивший мяч
